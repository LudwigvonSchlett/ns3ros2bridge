# ns3Ros2Bridge



## NS3-33

- [] Contient les scripts et sources nécessaires pour le fonctionnement du projet avec ns3
- [] Configurer et vérifier le fonctionnement de ns3 

```
cd ns3-33
sudo ./dependencies.bsh
./install.bsh
./test.bsh 

```

- [] Compiler de nouveaux changement

```
cd ns3-33
./compile.bsh
```

- [] Lancer le projet

```
cd ns3-33
sudo ./run.bsh
```
